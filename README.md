# Smart-Home-Automation-using_ESP32
ESP32-based IoT smart home automation and monitoring system for wireless appliance control, sensor monitoring, and remote access.
The Smart Home Monitoring System is an IoT-based project developed using ESP32 for real-time environmental monitoring inside homes. The system monitors air quality, temperature, and humidity using multiple sensors and displays the collected data on an OLED display while also enabling wireless monitoring through WiFi connectivity.

This project demonstrates embedded systems, IoT communication, sensor interfacing, and real-time monitoring applications.

---

## Features
- Real-time air quality monitoring using MQ135
- Temperature and humidity monitoring using DHT11
- OLED display for live sensor visualization
- WiFi-enabled IoT monitoring system
- Compact and low-power embedded design
- Continuous environmental data acquisition

---

## Components Used
- ESP32 Development Board
- MQ135 Air Quality Sensor
- DHT11 Temperature and Humidity Sensor
- OLED Display Module
- Jumper Wires
- Breadboard
- WiFi Network

---

## Working Principle
The ESP32 continuously collects environmental data from the MQ135 and DHT11 sensors. The acquired sensor values are processed and displayed on the OLED screen in real time.

Using WiFi connectivity, the ESP32 can also transmit sensor data for remote monitoring and future cloud integration. The system helps monitor indoor environmental conditions and can be expanded for smart automation applications.
